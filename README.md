# ```Needed```
*- Module Axios*
Install the Axios module by running the following command:
- Using npm:
```
npm install axios
```
- Using bower:
```
bower install axios
```
- Using yarn:
```
yarn add axios
```
- Using pnpm:
```
pnpm add axios
```
